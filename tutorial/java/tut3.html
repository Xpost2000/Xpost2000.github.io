For Loop

Now guys from Robotics. Remember the normal loop?

Yes again it's pretty much the same concept. The Syntax again is simple

for (Expression 1; Expression 2; Expression 3)

Execute code until one of the expressions are false

As you can see it is very basic logic and well can actually be used to replace

a While loop as it is less intensive by doing this

for (;;;)

Do things

this code would perform a task indefinitely within the forloop being the expressions are empty
