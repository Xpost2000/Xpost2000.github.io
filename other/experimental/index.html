    • This website isn't particularly interesting in terms of code, though for being produced with nothing but standard JS, HTML, and CSS it may be impressive in that front.

    It's designed with a pretty standard hash-router setup for an SPA. The animation is done the exact same way I would've done so with a game, with slight accomodations made to account for the retained UI paradigm that's on the web. The rest is done through CSS, which is actually really minimal. Most of the effects come from a looping video in the background as well as a screen-size tinting rectangle.

    Like my actual website, care was taken to make pages only take one screenful, so as to avoid breaking immersion.
