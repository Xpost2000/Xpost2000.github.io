Printf The Superior Printing Function

Printf stands for Formatted Printing! Not sure what formatting is?

say you created a language. Some symbols will stand for other things.

printf formats specific symbols. Here is the Symbol Table (The common ones anyways)

Format Code In English What It Does Example Usage
%d Decimal/Double Precision Number Allows decimal numbers to print use %f for floats printf("%d",3.3);
%i Integer Number Allows strictly integers to print (characters also work cause ASCII) printf("%i", 1);
%c Character Prints integers to the appropriate ASCII code printf("%c", 25); or printf("%c", 'a');
%b Boolean (Java exclusive) prints whether a Boolean is true or not printf("%b", true);
%s String (array of Characters) prints an entire string printf("%s","O Hello!");
%x A Hexadecimal Integer prints a number in hexadecimal printf("%x", 512 );

Didn't have enough of reading? Good here is another table for Escape Codes (Again common ones) Only works with ASCII Encoding

Escape Code In English What it Does Example Usage
\n (or use %c with 0x0a) Make A Newline Skips to the next line printf("yest \n test");
\t Make a Tab Adds a Tab to current line printf("\tTest");
\b Backspace As if you pressed backspace printf("Test\b");
\' Quotation Mark Adds Quotation Mark printf("\'Oswald\'");
\" Double Quotation Mark Adds two Quotation Marks printf("\"A Human\"");
\\ Backslash Adds Backslash openFile("C:\\MyPC\\Files\\abcd.txt");

Notice how Quotation Marks and Backslashes have their own escape code.

This is mainly because Quotation Marks are used to declare strings so it would be read as if

you were making a string literal (Which you are likely not) and gives an error. Backslashes have no real technical

influence, and are likely because Java is based of C which was a Unix made language. The Unix File Directory System worked like this

Instead of C:\Thing\test it would be C:\\Thing\\test

Back To Variables

There are many variable types but here the ones you need to know.

Type What it is
int a 4 bit numerical data type. Is 4 bits, suffices for most types of variables.
short 16 bit numerical data type. Is 16 bits, usage is tricky since you won't know the situations until you do.
long 32 bit numerical data type. Is 32 bits, usage is also tricky like shorts since it isn't easy to access most of the bits ( or use effectively at a beginner level).
char 1 bit numerical data type. Is 1 bit, typically used for ASCII characters, can also store standard numbers but not decimals.
String An array of characters. Everything that a char is also applies to string but bits depends on length (or amount of chars)
Boolean A variable that can only be set to the value true or false. Represented by 1 or 0

Like most programming languages, Java provides the essential data types and leaves you to do things with.

Like for example you can use a int to determine the health an enemy has or something like attack damage.
